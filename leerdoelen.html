Ik heb deze week het html schrijven geleerd en hoe ik die vormen en kleuren kan geven met behulp van css, waardoor ik ook heb geleerd hoe ik gebruik moet maken van verschillende selectors om de elementen in de html aan te sturen. Ik heb voor mijn opdracht ook wat class en id attributen gebruikt en heb hierdoor geleerd dat een class meerdere keren gebruikt kan worden om verschillende elementen dezelfde styl te geven en het id attribuut krijgt voorrang als ze beiden worden ingezet voor een element. Parent-child relaties werken zoals het bij ons ook hoort de kinderen luisteren naar hun ouders. Zo heb ik met div wat elementen aangestuurd in mijn opdracht om alles op de goede plaats te krijgen in mijn opdracht.

De komende periode wil ik het html en css nog beter onder de knie krijgen en flink oefenen op de styling doormiddel van de elementen juist aan te sturen door het juist gebruiken van de selectoren. Ook wil ik goed met github en commands leren omgaan om zo goed mogelijk te kunnen werken aan mijn wekelijkse opdrachten, zodat ik goed ben voorbereid voor de hackaton. Ik wil zoveel mogelijk leren om een goede developer te kunnen worden. Ik weet nog niet precies welke kant ik op wil gaan, maar wel dat als ik genoeg kennis heb in ieder geval mijn eigen app voor muziekproductie wil maken.
